# cara pakai
```
    buat database dengan nama db_fastbites
    composer install
    php artisan migrate:fresh --seed
    php artisan storage:link
    php artisan migrate
    download node.js
    npm install
    npm run dev dab php artisan serve
    http://localhost:8000/
    http://localhost:8000/admin
```
